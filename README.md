## 3.使用方法
### 3-1.タスクの追加
| 項目 | 内容 |
|---|---|
| タイトル | タスクのタイトルを入力します。 |
| 説明 | タスクの説明を入力します。任意での入力です。 |
| 優先度 | タスクの優先度、高い、普通、低いの3段階で設定できます。 |
| ステータス | 追加時のタスクの状況を設定できます。未着手、進行中、完了の3段階で設定できます。 |
| 期限 | タスクの締め切りを日付形式で設定できます。 |
| タグ | タグを設定します。カンマ「,」で区切ることが出来ます。 |
